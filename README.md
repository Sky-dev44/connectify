# 💬 FullStack Chat App

A real-time full-stack chat application built with the MERN stack, featuring instant messaging powered by WebSockets.
---

## ✨ Features

- 🔐 User authentication (Login / Signup)
- 💬 Real-time messaging with WebSockets
- 🔔 Message notification sound
- 🟢 Online user presence indicator
- 🎨 Multiple themes powered by DaisyUI
- 📱 Fully responsive design

---

## 🛠️ Tech Stack

**Frontend:**
- React (Vite)
- Tailwind CSS
- DaisyUI
- Zustand (state management)
- Socket.io-client

**Backend:**
- Node.js
- Express.js
- MongoDB + Mongoose
- Socket.io
- JSON Web Tokens (JWT)

---

## 📦 Installation & Setup

### 1. Install dependencies

# Backend
cd backend
npm install

# Frontend
cd ../frontend
npm install


### 2. Run the app

# Run backend
cd backend
npm run dev

# Run frontend
cd frontend
npm run dev
